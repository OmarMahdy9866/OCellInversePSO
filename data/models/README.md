# PLAXIS Base Model

Place the calibrated starting PLAXIS 3D model here as:

- `OCell_base.p3d`

The model should already contain:

- the soil material sets named to match `config/soil_profile.yaml`
- the O-Cell loading phases named to match `config/project.yaml`
- output nodes that can be resolved by the `ocell.upper_node` and `ocell.lower_node` settings
